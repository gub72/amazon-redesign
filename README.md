## Sobre o Projeto

Um site de e-commerce full-stack inspirado no site da Amazon.

<img width="1493" alt="amazon-redesign" src="https://user-images.githubusercontent.com/102174805/187029436-ee2149f4-43db-4145-b67d-a16bb4317357.png">

**Funcionalidades Principais:**
* Criar/Entrar em uma conta.
* Aplicação de Página Única (SPA - tempo de resposta instantâneo).
* Responsividade para Desktop/Mobile.
* Adicionar itens ao carrinho.
* Realizar pagamentos.
* Armazenamento de informações do usuário para contas individuais.

Embora não emule todas as funcionalidades do site real, ele mantém a maioria delas. É um projeto excepcional, utilizável em todos os dispositivos, detecta usuários logados e muito mais. Também é construído com REACT & REDUX! Espero que você goste. Clique [aqui](#) para ver o projeto ao vivo na web.

## Começando

### Clonar Localmente
- Quando aberto, navegue até o nível superior esquerdo do projeto, um botão verde de download do código estará visível no lado direito.
- Selecione a opção "Download Zip" no menu suspenso.
- Assim que o download for concluído, você poderá acessar meu projeto localmente.
